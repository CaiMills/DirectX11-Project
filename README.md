# Controls for the Artefact;
## View Mode;
- F1: Solid/Fill
- F2: Wireframe
## FreeCam Movement;
- I: Forwards
- K: Backwards
- J: Left
- L: Right
## Add Force;
- W: Fowards
- S: Backwards
- A: Left
- D: Right
- Z: Up
- X: Down
## Add Velocity
- Arrow Up: Fowards
- Arrow Down: Backwards
- Arrow Left: Left
- Arrow Right: Right
- Page Up: Up
- Page Down: Down
## Other 
- Q - Applies Quaternion Force 
- G - Turn Gravity On
- H - Turn Gravity Off
